lets learn about margin,border,padding

padding is the space directly around your content it is used to add distance beetween your content and the border. you can interact with it in css by using the "padding:00px" function to add more or less adjust the tag "00px" adding numbers like so "100px". you can also add padding to specific sides to do this add a side to the "padding" such as "padding-top".

border this is located around the padding element, the main use of border is to add colour to the outside of your element you can do this with the border-color:"green" function you can also choose the style of border with border-style:"solid" function aswell as the thickness of the border with border-width:"20px".

margin is the outside of the border function the use of margin is for isolating your element if you dont want other things directely around it you use margin:00px to set the margin, basically making an invisible wall around your element.
