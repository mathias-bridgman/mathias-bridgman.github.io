explaing control flow and loops

control flow is the way that a computer runs code like how people do things through the day like making then eating breakfast, then showering etc the pattern of what it does every time. there are three main types if/else statements which tests a boolean if its true it runs a ceartain peice of code and if its false it returns another this happens because the control list has a checklist if you will of everything it needs to do in order.
loops are a type of code in javascript that loops and repeats things the classic loop goes.
"for(let i=0 i <= 0 i++){"some code you want to run multiple(times)"}".
with a loop you can make it run as many times as you want by changing the value "i < 0" to something like "i <= 5" this would make the code run 5 times until the i is equal to 5, the i++ means that at the end of the code running it adds 1 to i so you can track how many times its run.
